About THE PROJECT

In this project, we will explore sentiment analysis, which can be beneficial for companies to understand the market and improve their products and for consumers to make informed decisions when purchasing goods. We will investigate and optimize various machine learning models, including the State-of-the-Art (SOTA) model, trained using 3.6 million Amazon reviews to perform sentiment analysis.

Amazon is one of the largest e-commerce platforms in the world.

Understanding the general sentiment of the many reviews requires a fast and accurate model.

We aim to make a machine learning application that uses natural language processing (NLP) algorithms, i.e., sentiment analysis to analyze Amazon reviews and classify them as positive or negative based on their sentiment. This application will provide a classification summary of the overall sentiment for a product, allowing users to quickly gauge customer sentiment and make informed decisions. Our user demographics for this application are companies and researchers who need to analyze large volumes of reviews. Indeed, since the structure of reviews are largely similar across the internet, our trained model can also be fed texts coming from other sites as well.

We have successfully acquired a 94.4% test accuracy using a CNN-LSTM model, tested on 400,000 reviews.

The Pipeline

01

The Data

3.6 millions datapoints on the training set, containing review texts and binary labels, classifying each datapoints to either positive or negative. 400,000 datapoints on the test set with the same format

02

Cleaning the Data

Raw reviews are dirty. We can remove irrelevant parts to optimise accuracy and speed. This includes: URLs, stopwords, special characters, punctuations, etc.

03

Extracting Features

The cleaned texts need to somehow be transformed into a set of numerical features such that the model can work with it. In this case, the features will be an array of weighted word counts for each word in a datapoint

04

Model Exploration

We select conventional and SOTA models, optimize them, and analyze their performance.
